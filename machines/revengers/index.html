期待値の目安

等価交換：通常450G〜 / リセット時100G〜 / 東卍チャンス3スルー時100G〜

5.6枚交換：通常470G〜 / リセット時130G〜

現金投資（5.0枚交換）：通常520G〜 / リセット時170G〜

AT間1190G+α天井（AT確定）と東卍チャンス4スルー天井の複合機種。450G以降は安定して期待値プラスで、3スルー以上ならG数100以上から狙えるなどスルー狙いとの合わせ技で浅い条件でもプラス域に入ります。

朝一・リセット情報

リセット判別方法：朝一のG数表示が0スタートかどうかと、東卍チャンスのスルー回数が0回にリセットされているかを確認。前任者のスルー数がリセットされていれば設定変更濃厚です。

リセット恩恵：AT間天井が1190G+α→900G+αに短縮、初回周期も最大200ptに圧縮されます。

朝一の狙い目：リセット確認後は等価100G〜から狙えます。リセットなしの据え置き台では、東卍チャンスのスルー数が引き継がれているため、深いスルー数の台も優秀な狙い目です。

狙い目の根拠

スマスロ東京リベンジャーズはAT間1190G+αが天井でAT確定。等価450G〜が狙い目ラインで、5.6枚交換なら470G、現金投資なら520G〜が目安です。機械割97.8%（設定1）で天井狙いの期待値は十分あり、450G時点で残り天井距離は約740Gに圧縮されるため期待収支がプラス側に乗ります。

東卍チャンス（CZ）4スルーでAT確定という強力な天井があり、スルーが積み重なるほど狙い目ラインが浅くなります。3スルー以上の台は特に積極的に狙いましょう。3スルー時はG数100以上、2スルー時はG数250以上、1スルー時はG数350以上が目安となり、スルーが進むほど0Gスタートに近い感覚で取れます。

リセット時は天井が900G+αに短縮され、初回周期も最大200ptに圧縮されます。設定変更恩恵が強いため、朝一の据え置き判別が取れる店舗ではリセット濃厚台を100Gほどから狙うことも有効です。

ヤメ時の判断

AT/ボーナス後はサブ液晶タッチでセリフを確認して即やめ。ノイズ演出中（約50G）はリベンジ発生の可能性があるため消えるまでフォロー。赤セリフなら周期当選まで続行。

AT/ボーナス終了後はサブ液晶タッチでセリフを確認して即やめが基本です。引き戻しのノイズ演出（約50G）が見えた場合は消えるまでフォローし、赤セリフが出たら次周期当選まで続行してください。引き戻しゾーン（特殊ステージ）滞在中は継続必須で、ここでヤメてしまうと一番おいしい部分を逃します。

スルー回数はヤメた後も台に引き継がれます。スルー数が多い台を離席すると他のプレイヤーに有利な状態を渡すことになる点も考慮してください。3スルー以上をすでに溜めた台で「あと1スルー進めば天井確定」という状況なら、サブ液晶の引き戻し示唆を待たずにそのまま継続する選択肢もあります。

立ち回りのコツ

天井狙い・スルー狙い・リセット狙いの3本柱。スルー3回以上の台は積極的に打てます。周期5周期目以降も狙い目です。

スルー回数の確認が最優先事項です。データカウンターでスルー数が確認できる場合は必ずチェックしてから着席を判断しましょう。スルーは通常AT間天井1190G+αとは別系統で蓄積するため、AT間が浅くてもスルー回数が進んでいれば期待値が成立します。

純増は通常AT約3.2枚・上位AT約8.0枚/Gと、上位ATに入った際のリターンが大きい機種です。4スルー以上はG数に関わらず最優先で確保してください。共通ベル（15枚役）に設定差があり、設定1で1/99.3、設定6で1/77.1なので、高設定示唆が出た台はそのままポチポチくんで設定推測に切り替えると無駄打ちが減ります。

設定判別のポイント

共通ベル（15枚役）に設定差があり、設定1で1/99.3、設定6で1/77.1です。通常時のみカウント推奨。ポチポチくんで追跡できます。
